# 🛡️ Checklist Obligatorio de Verificación (Pre-Deployment)

Antes de confirmar cualquier cambio o realizar un deployment, es OBLIGATORIO verificar los siguientes puntos en la aplicación:

1.  **Fuentes de pago:**
    -   [ ] Verificar en ajustes que aparecen al menos: **Efectivo**, **Tarjeta Débito** y **Tarjeta Crédito**.
2.  **Presupuesto mensual:**
    -   [ ] Verificar que el presupuesto configurado (ej. 6.000.000) no cambie tras recargar o navegar.
3.  **Gastos fijos:**
    -   [ ] Verificar que la lista de gastos fijos configurada se mantenga visible.
4.  **Ingresos recurrentes:**
    -   [ ] Verificar que los ingresos recurrentes se mantengan visibles.
5.  **Formularios de Gasto:**
    -   [ ] Abrir "Nuevo Gasto" y "Gasto Rápido" y confirmar que el desplegable de **Cuenta** muestra todas las opciones disponibles (Efectivo, Débito, Crédito).

// turbo
**IMPORTANTE**: Si alguno de estos puntos falla, la actualización se considera FALLIDA. Se debe revertir o corregir antes de notificar al usuario.
